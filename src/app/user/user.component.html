Instruções para Gravação dos Áudios

  • Para o correto funcionamento desta aplicação, utilizar o Google Chrome.

  • O usuário deve estar próximo o suficiente para que o microfone do notebook/desktop capte a voz adequadamente.

  • Ao realizar a gravação, escute-a e verifique se a frase foi pronunciada corretamente e a voz está audível. Caso contrário, tente gravar novamente.

  • As informações solicitadas abaixo devem ser preenchidas corretamente, pois a partir delas será possível identificar e classificar cada áudio gravado.

Informações do Locutor

Não necessariamente a cidade onde reside ou nasceu, mas de onde seu sotaque é caracterizado.
